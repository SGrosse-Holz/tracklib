1""" 

2Likelihood (i.e. "proper") MSD fitting 

3 

4In fact, we should talk about autocorrelation fitting, as that is the central 

5object (since it defines the associated Gaussian Process). However, in the 

6context of single particle tracking it seems that people prefer MSDs. These are 

7equivalent viewpoints, as outlined below. This module works with ACFs, but is 

8well set up for translation to MSDs. 

9 

10The basic idea is that as soon as we assume steady state, we can immediately 

11define a covariance operator Σ(t, t') := γ(|t-t'|) from an autocorrelation 

12function. We can then use this covariance operator to marginalize to time 

13points where we have data, thus obtain a covariance matrix, and calculate a 

14likelihood of observing those data. 

15 

16The above paragraph is vague in that it does not state what autocorrelation 

17function we are actually talking about, and there are two important versions: 

18 + if the trajectory itself is sampled from a steady state (e.g. distance of 

19 points on a polymer) then we use the autocorrelation function of the actual 

20 data points. The information contained therein is exactly equivalent to the 

21 MSD + steady state variance. We will call this a steady state of order 0. 

22 The relation between MSD μ(k) and steady state variance V, and process 

23 autocorrelation γ(k) is 

24 μ(k) = 2*( γ(0) - γ(k) ), V = γ(0) 

25 + in many cases (e.g. sampling a diffusing particle's position) the trajectory 

26 itself will not come from a steady state process, but its displacements do. 

27 In this case, which we call steady state of order 1, the autocorrelation 

28 function that we wish to fit is the one of the displacement process. In this 

29 case, the information in autocorrelation (of the displacement process) γ(k) 

30 and MSD μ(k) is exactly the same, and we have 

31 γ(k) = 1/2*( μ(k+1) - μ(k-1) + 2μ(k) ). 

32 For inversion note that μ(-1) = μ(1) = γ(0) and μ(0) = 0. 

33 + theoretically, one could imagine higher order steady states, i.e. the n-th 

34 order difference of the trajectory comes from a steady state. We trust that 

35 this case is of minor importance and can thus safely be ignored (for now). 

36Note the provided functions to convert between (MSD, V) <--> ACF for order 0 

37and MSD <--> ACF for order 1 

38 

39At the end of the day, it seems that for steady states of order 0, the natural 

40description is via the autocorrelation function, while for steady states of 

41order 1 it is the MSD. Note however that a fair bit of (my) intuition is 

42associated with MSDs. 

43 

44This module's main interface are the subclasses of `Fit`, which all implement 

45different schemes for fitting MSDs/ACFs to a given data set. `ds_logL` provides 

46the likelihood function for a given ACF on a given data set. 

47 

48There is a model free approach that fits the MSD with a cubic spline, thus 

49giving an analytically meaningful MSD (i.e. one that actually defines a 

50Gaussian process) without having to assume a model. In many use cases, this 

51even has a (significantly) higher likelihood than the empirical MSD calculated 

52by `analysis.MSD`. In any case it seems like a better representation, since it 

53has way fewer parameters. In fact one can apply Bayesian model selection to 

54find the best number of spline knots. 

55 

56Other than the model free version, so far we have 

57 + NPXFit: Noise + Powerlaw + X. This fits a powerlaw to the beginning of the 

58 MSD and compensates everything at long times with a spline, then adds some 

59 constant offset for localization error. 

60 + RouseFit, RouseFitDiscrete: might be moved eventually. Fit two-locus Rouse 

61 models, either with an infinite, continuous chain, or a model like 

62 `tracklib.models.Rouse`.

104 def wrap(dt): 

105 # Preproc 

106 dt = np.abs(np.asarray(dt))

108 if was_scalar: 

109 dt = np.array([dt])

117 msd = msd[0]

121def msd2C(msd, ti, ss_order): 

122 # msd can be function msd(t) or, for integer ti, an array msd[t]. For 

123 # ss_order == 0, we expect msd(np.inf) or msd[-1] to give the limit of the 

124 # MSD for t --> infinity. 

125 if ss_order == 0: 

126 if callable(msd): 

127 return 0.5*( msd(np.inf) - msd(ti[:, None] - ti[None, :]) ) 

128 else: 

129 return 0.5*( msd[-1] - msd[np.abs(ti[:, None] - ti[None, :])] ) 

130 elif ss_order == 1: 

131 if callable(msd): 

132 return 0.5*( msd(ti[1:, None] - ti[None, :-1]) + msd(ti[:-1, None] - ti[None, 1: ]) 

133 - msd(ti[1:, None] - ti[None, 1: ]) - msd(ti[:-1, None] - ti[None, :-1]) ) 

134 else: 

135 return 0.5*( msd[ti[1:, None] - ti[None, :-1]] + msd[ti[:-1, None] - ti[None, 1: ]] 

136 - msd[ti[1:, None] - ti[None, 1: ]] - msd[ti[:-1, None] - ti[None, :-1]] ) 

137 else: 

138 raise ValueError(f"Invalid stead state order: {ss_order}")

151def _GP_core_logL(C, x): 

152 # likelihood calculation 

153 with np.errstate(under='ignore'): 

154 s, logdet = np.linalg.slogdet(C) 

155 if s <= 0: 

156 raise BadCovarianceError("Covariance not positive definite. slogdet = ({}, {})".format(s, logdet)) 

157 

158 try: 

159 xCx = x @ linalg.solve(C, x, assume_a='pos') 

160 except (FloatingPointError, linalg.LinAlgError) as err: 

161 GP_vprint(3, f"Problem when inverting covariance, even though slogdet = ({s}, {logdet})") 

162 GP_vprint(3, type(err), err) 

163 raise BadCovarianceError("Inverting covariance did not work") 

164 

165 return -0.5*(xCx + logdet) - len(C)*LOG_SQRT_2_PI

167def GP_logL(trace, ss_order, msd, mean=0): 

168 # mean: mean if ss=0, drift if ss=1 

169 ti = np.nonzero(~np.isnan(trace))[0] 

170 

171 if ss_order == 0: 

172 X = trace[ti] - mean 

173 elif ss_order == 1: 

174 X = np.diff(trace[ti]) - mean*np.diff(ti) 

175 else: 

176 raise ValueError(f"Invalid stead state order: {ss_order}") 

177 

178 C = msd2C(msd, ti, ss_order) 

179 return _GP_core_logL(C, X)

181def _GP_logL_for_parallelization(params): 

182 return GP_logL(*params)

184def ds_logL(data, ss_order, msd_ms): 

185 # msd_ms : list of tuples: d*[(msd, m)] 

186 d = data.map_unique(lambda traj : traj.d) 

187 if len(msd_ms) != d: 

188 raise ValueError(f"Dimensionality of MSD ({len(msd_ms)}) != dimensionality of data ({d})") 

189 

190 # Convert msd to array, such that parallelization works 

191 Tmax = np.max(list(map(lambda traj : traj.T, data))) 

192 dt = np.arange(Tmax) 

193 array_msd_ms = [[msd(dt), m] for msd, m in msd_ms] 

194 if ss_order == 0: 

195 for dim, (msd, m) in enumerate(msd_ms): 

196 array_msd_ms[dim][0] = np.append(array_msd_ms[dim][0], msd(np.inf)) 

197 

198 job_iter = itertools.chain.from_iterable((itertools.product((traj[:][:, dim] for traj in data), 

199 [ss_order], [msd], [m], 

200 ) 

201 for dim, (msd, m) in enumerate(array_msd_ms) 

202 )) 

203 

204 return np.sum(list(parallel._umap(_GP_logL_for_parallelization, job_iter)))

250 def constraint_Cpositive(self, params): 

251 msd, _ = self.params2msdm(params) 

252 min_ev_okay = 1 - np.cos(np.pi/self.T) # white noise min ev 

253 min_ev = np.min(np.linalg.eigvalsh(msd2C(msd, np.arange(self.T), self.ss_order)))

258 def _penalty(self, params): 

259 x = np.inf 

260 for constraint in self.constraints: 

261 x = min(constraint(params), x) 

262 if x <= 0: 

263 return -1 # unfeasible 

264 

265 if x >= 1: 

266 return 0 

267 else: 

268 with np.errstate(over='raise', under='ignore'): 

269 try: 

270 return min(np.exp(1/np.tan(np.pi*x)), self.max_penalty) 

271 except: 

272 return self.max_penalty

316 def min_target(params, just_return_full_params=False, do_fixing=do_fixing): 

317 if do_fixing: 

318 params = fixer(params) 

319 if just_return_full_params: 

320 return params 

321 

322 penalty = self._penalty(params) 

323 if penalty < 0: # infeasible 

324 return self.max_penalty 

325 else: 

326 return -ds_logL(self.data, 

327 self.ss_order, 

328 self.params2msdm(params), 

329 ) \ 

330 - self.logprior(params) \ 

331 + penalty \ 

332 - offset 

333 

334 return min_target

336 def run(self, 

337 init_from = None, # dict as returned by this function 

338 optimization_steps=('simplex',), 

339 maxfev=None, 

340 fix_values = None, # list of 2-tuples (i, val) to fix params[i] = val 

341 full_output=False, 

342 show_progress=False, 

343 verbosity=None, # temporarily overwrite verbosity settings 

344 ): 

345 if verbosity is not None: 

346 tmp = self.verbosity 

347 self.verbosity = verbosity 

348 verbosity = tmp 

349 self.data.restoreSelection(self.data_selection) 

350 for step in optimization_steps: 

351 assert type(step) is dict or step in {'simplex', 'gradient'} 

352 

353 # Initial values 

354 if init_from is None: 

355 p0 = self.initial_params() 

356 total_offset = self.initial_offset() 

357 else: 

358 p0 = deepcopy(init_from['params']) 

359 total_offset = -init_from['logL'] 

360 

361 # Adjust for fixed values 

362 ifix = [i for i, _ in self.expand_fix_values(fix_values)] 

363 bounds = deepcopy(self.bounds) 

364 for i in sorted(ifix, reverse=True): 

365 del bounds[i] 

366 p0[i] = np.nan 

367 p0 = p0[~np.isnan(p0)] 

368 

369 # Set up progress bar 

370 bar = tqdm(disable = not show_progress) 

371 def callback(x): 

372 bar.update() 

373 

374 # Go! 

375 all_res = [] 

376 with np.errstate(all='raise'): 

377 for istep, step in enumerate(optimization_steps): 

378 if step == 'simplex': 

379 options = {'fatol' : 0.1, 'xatol' : 0.01} 

380 self.vprint(3, "Note: why is `xatol = 0.01` a good choice? / do we need it?") 

381 if maxfev is not None: 

382 options['maxfev'] = maxfev 

383 kwargs = dict(method = 'Nelder-Mead', 

384 options = options, 

385 bounds = bounds, 

386 callback = callback, 

387 ) 

388 elif step == 'gradient': 

389 options = {} 

390 if maxfev is not None: 

391 options['maxfev'] = maxfev 

392 kwargs = dict(method = 'L-BFGS-B', 

393 options = options, 

394 bounds = bounds, 

395 callback = callback, 

396 ) 

397 else: 

398 kwargs = dict(callback = callback, 

399 bounds = bounds, 

400 ) 

401 kwargs.update(step) 

402 

403 min_target = self.get_min_target(offset=total_offset, fix_values=fix_values) 

404 try: 

405 fitres = optimize.minimize(min_target, p0, **kwargs) 

406 except BadCovarianceError as err: 

407 self.vprint(2, "BadCovarianceError:", err) 

408 fitres = lambda: None 

409 fitres.success = False 

410 

411 if not fitres.success: 

412 self.vprint(1, f"Fit (step {istep}: {step}) failed. Here's the result:") 

413 self.vprint(1, fitres) 

414 raise RuntimeError("Fit failed at step {:d}: {:s}".format(istep, step)) 

415 else: 

416 all_res.append(({'params' : min_target(fitres.x, just_return_full_params=True), 

417 'logL' : -(fitres.fun+total_offset), 

418 }, fitres)) 

419 p0 = fitres.x 

420 total_offset += fitres.fun 

421 

422 bar.close() 

423 if verbosity is not None: 

424 self.verbosity = verbosity 

425 

426 if full_output: 

427 return all_res 

428 else: 

429 return all_res[-1][0]

589 and self.restart_on_better_point_estimate 

590 ): 

591 raise Profiler.FoundBetterPointEstimate

593 def run_fit(self, is_new_point_estimate=True, 

594 **fit_kw, 

595 ): 

596 self.run_count += 1 

597 if self.run_count > self.max_fit_runs: 

598 raise StopIteration(f"Ran out of likelihood evaluations (max_fit_runs = {self.max_fit_runs})") 

599 

600 if 'init_from' not in fit_kw: 

601 fit_kw['init_from'] = self.best_estimate 

602 

603 if self.point_estimate is None and fit_kw['init_from'] is None: # very first fit, so do simplex --> (gradient) 

604 res = self.fit.run(optimization_steps = ('simplex',), 

605 **fit_kw, 

606 ) 

607 try: # try to refine 

608 fit_kw['init_from'] = res 

609 fit_kw['show_progress'] = False 

610 res = self.fit.run(optimization_steps = ('gradient',), 

611 **fit_kw, 

612 ) 

613 except RuntimeError: # okay, this didn't work, whatever 

614 pass 

615 else: # we're starting from somewhere known, so start out trying to move by gradient, use simplex if that doesn't work 

616 try: 

617 res = self.fit.run(optimization_steps = ('gradient',), 

618 verbosity=0, 

619 **fit_kw, 

620 ) 

621 if fit_kw['init_from'] is not None and self.likelihood_significantly_greater(fit_kw['init_from'], res): 

622 # This happens sometimes, who knows why 

623 # Update: was a bug, but can't hurt to keep this mechanism around 

624 raise RuntimeError 

625 except RuntimeError: 

626 self.vprint(2, "Gradient fit failed, using simplex") 

627 res = self.fit.run(optimization_steps = ('simplex',), 

628 **fit_kw, 

629 ) 

630 

631 if fit_kw['init_from'] is not None and self.likelihood_significantly_greater(fit_kw['init_from'], res): 

632 self.vprint(2, ("Both simplex and gradient fits gave worse than initial value: " 

633 f"{res['logL']} < {fit_kw['init_from']['logL']}. Using initial instead")) 

634 res = fit_kw['init_from'] 

635 

636 if is_new_point_estimate: 

637 self.point_estimate = res 

638 else: 

639 self.ress[self.iparam].append(res) 

640 self.check_point_estimate_against(res) 

641 

642 if self.bar is not None: 

643 self.bar.update()

647 def find_closest_res(self, val, direction=None): 

648 ress = self.ress[self.iparam] + [self.point_estimate] 

649 

650 values = np.array([res['params'][self.iparam] for res in ress]) 

651 if val in values: 

652 i = np.argmax([res['logL'] for res in ress if res['params'][self.iparam] == val]) 

653 return ress[i] 

654 

655 if self.bracket_strategy[self.iparam]['multiplicative']: 

656 distances = np.abs([np.log(val/res['params'][self.iparam]) for res in ress]) 

657 else: 

658 distances = np.abs([val-res['params'][self.iparam] for res in ress]) 

659 

660 if direction is not None: 

661 distances[np.sign(values - val) != direction] = np.inf 

662 

663 min_dist = np.min(distances) 

664 

665 i_candidates = np.nonzero(distances < min_dist+1e-10)[0] # We use bisection, so usually there will be two candidates 

666 ii_cand = np.argmax([ress[i]['logL'] for i in i_candidates]) 

667 

668 return ress[i_candidates[ii_cand]]

670 def profile_likelihood(self, value, init_from='closest'): 

671 if self.profiling: 

672 if init_from == 'closest': 

673 init_from = self.find_closest_res(value) 

674 

675 self.run_fit(init_from = init_from, 

676 fix_values = [(self.iparam, value)], 

677 is_new_point_estimate = False, 

678 ) 

679 else: 

680 new_params = self.point_estimate['params'].copy() 

681 new_params[self.iparam] = value 

682 minus_logL = self.min_target_from_fit(new_params) 

683 

684 if self.bar is not None: 

685 self.bar.update() 

686 

687 self.ress[self.iparam].append({'logL' : -minus_logL, 'params' : new_params}) 

688 self.check_point_estimate_against(self.ress[self.iparam][-1]) 

689 

690 return self.ress[self.iparam][-1]['logL']

692 def iterate_bracket_point(self, x0, pL, direction, 

693 step = None, 

694 ): 

695 self.expand_bracket_strategy() 

696 if step is None: 

697 step = self.bracket_strategy[self.iparam]['step'] 

698 

699 bracket_param = 1 if self.bracket_strategy[self.iparam]['multiplicative'] else 0 

700 p = x0 

701 pL_thres = self.point_estimate['logL'] - self.LR_target 

702 while pL > pL_thres: 

703 self.vprint(3, "bracketing: {:.3f} > {:.3f} @ {}".format(pL, pL_thres, p)) 

704 

705 # Check identifiability cutoff 

706 ib = int((1+direction)/2) 

707 if bracket_param > self.bracket_strategy[self.iparam]['nonidentifiable_cutoffs'][ib]: 

708 self.vprint(2, "{} edge of confidence interval is non-identifiable".format('left' if direction == -1 else 'right')) 

709 p = self.fit.bounds[self.iparam][ib] 

710 pL = np.inf 

711 break 

712 

713 # Update position 

714 if self.bracket_strategy[self.iparam]['multiplicative']: 

715 bracket_param *= step 

716 p = x0 * bracket_param**direction 

717 else: 

718 bracket_param += step 

719 p = x0 + direction*bracket_param 

720 

721 # Update profile likelihood 

722 pL = self.profile_likelihood(p) 

723 else: 

724 self.vprint(3, "bracketing: {:.3f} < {:.3f} @ {}".format(pL, pL_thres, p)) 

725 

726 return p, pL

728 def initial_bracket_points(self): 

729 self.expand_bracket_strategy() 

730 a, a_pL = self.iterate_bracket_point(self.point_estimate['params'][self.iparam], self.point_estimate['logL'], direction=-1) 

731 b, b_pL = self.iterate_bracket_point(self.point_estimate['params'][self.iparam], self.point_estimate['logL'], direction= 1) 

732 return (a, a_pL), (b, b_pL)

734 def solve_bisection(self, bracket, bracket_pL): 

735 c = np.mean(bracket) 

736 c_pL = self.profile_likelihood(c) 

737 

738 a_fun, b_fun = self.point_estimate['logL'] - bracket_pL - self.LR_target 

739 c_fun = self.point_estimate['logL'] - c_pL - self.LR_target 

740 i_update = 0 if a_fun*c_fun > 0 else 1 

741 assert [a_fun, b_fun][1-i_update]*c_fun <= 0 

742 

743 bracket[i_update] = c 

744 bracket_pL[i_update] = c_pL 

745 

746 if np.all([self.LR_interval[0] < LR < self.LR_interval[1] for LR in self.point_estimate['logL'] - bracket_pL]): 

747 return np.mean(bracket) 

748 else: 

749 return self.solve_bisection(bracket, bracket_pL)

751 @restart_if_better_pe_found 

752 def find_single_MCI(self, iparam): 

753 self.iparam = iparam 

754 if self.point_estimate is None: 

755 raise RuntimeError("Need to have a point estimate before calculating confidence intervals") 

756 

757 (a, a_pL), (b, b_pL) = self.initial_bracket_points() 

758 m = self.point_estimate['params'][self.iparam] 

759 m_pL = self.point_estimate['logL'] 

760 

761 roots = np.array([np.nan, np.nan]) 

762 if a_pL == np.inf: 

763 roots[0] = a 

764 if b_pL == np.inf: 

765 roots[1] = b 

766 

767 for i, (bracket, bracket_pL) in enumerate(zip([(a, m), (m, b)], [(a_pL, m_pL), (m_pL, b_pL)])): 

768 if np.isnan(roots[i]): 

769 roots[i] = self.solve_bisection(np.asarray(bracket), np.asarray(bracket_pL)) 

770 if i == 0: 

771 self.vprint(2, f"Found left edge, now searching right one @ {roots[i]}") 

772 

773 self.vprint(2, f"found CI = {roots} (point estimate = {m}, iparam = {self.iparam})\n") 

774 return m, roots

776 def find_MCI(self, iparam='all', 

777 show_progress=False, 

778 ): 

779 if show_progress and self.bar is None: 

780 self.bar = tqdm() 

781 

782 if self.point_estimate is None: 

783 self.vprint(2, "Finding initial point estimate ...") 

784 self.run_fit(show_progress=show_progress) 

785 old_point_estimate = self.point_estimate 

786 

787 self.vprint(2, "initial point estimate: params = {}, logL = {}\n".format(self.point_estimate['params'], 

788 self.point_estimate['logL'], 

789 )) 

790 

791 n_params = len(self.point_estimate['params']) 

792 mcis = np.empty((n_params, 3), dtype=float) 

793 mcis[:] = np.nan 

794 

795 if iparam == 'all': 

796 fixed = [i for i, _ in self.fit.fix_values] 

797 iparams = np.array([i for i in range(n_params) if i not in fixed]) 

798 else: 

799 iparams = np.asarray(iparam) 

800 if len(iparams.shape) == 0: 

801 iparams = np.array([iparams]) 

802 

803 while True: # limited by max_restarts_per_parameter 

804 for iparam in iparams: 

805 self.run_count = 0 

806 self.vprint(2, f"starting iparam = {iparam}") 

807 m, ci = self.find_single_MCI(iparam) 

808 mcis[iparam, :] = m, *ci 

809 

810 if (self.likelihood_significantly_greater(self.best_estimate, old_point_estimate) 

811 and self.restart_on_better_point_estimate): 

812 self.vprint(2, f"Point estimate was updated while we calculated confidence intervals, so restart") 

813 self.vprint(2, "new best logL = {:.3f} > {:.3f} = old point estimate logL\n".format(self.best_estimate['logL'], 

814 old_point_estimate['logL'])) 

815 self.point_estimate = self.best_estimate # just in case it's not yet 

816 else: 

817 break 

818 

819 if show_progress and self.bar is not None: 

820 self.bar.close() 

821 self.bar = None 

822 

823 self.vprint(2, "Done\n") 

824 

825 if len(iparams) == 1: 

826 return mcis[iparams[0]] 

827 else: 

828 return mcis
